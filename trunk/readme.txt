Welcome to the OpenSign Project!

If you are looking for the sources of the OpenSign server, please find the
current version at:  src/OpenSignServer/trunk

The client applications are located at: src/OSSClientTools, whereas the only
released client application is the OSSJClient, which source code can be accessed
at: src/OSSClientTools/java/OSSJClient

For further details, please have a look at the readme-files in the root 
directory of the application you are interested in.
